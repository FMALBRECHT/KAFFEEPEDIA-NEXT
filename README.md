Next.js + MDX codebase for the full Kaffeepedia platform: Enzyklopädie, KAFFEEDUCTATION, Coffee Science, and the scientific document system. Includes dynamic MDX routing, KaTeX math, KaTeX styling, and the TUM/Apple design system.
# KAFFEEPEDIA-NEXT

Official Next.js + MDX codebase for **Kaffeepedia** — 
The World’s Coffee Encyclopedia & University.

This platform powers:
- The Enzyklopädie
- KAFFEEDUCTATION
- Coffee Biology / Chemistry / Physics
- MDX scientific documents (KaTeX math)
- Dynamic routes: /docs/[slug]
